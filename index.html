Chess Piece Image Classifier

I created an image classifier that takes image of a chess piece and identifies the name and color. I collected the approximately 200 images per piece and created the model using transfer learning with fastai on google colab. I am currently working on the web interface, using Flask, through which the model can accept image. For the next phase, I would work on processing image of complete chess board and build a chess engine to predict the next best move.
