Chess Piece Image Classifier

I created an image classifier that takes image of a chess piece and identifies the name and color. I collected approximately 200 images per piece and created the model using transfer learning with fastai on google colab. I am currently working on the web interface, using Flask, through which the model can accept image. For the next phase, I would work on processing image of complete chess board and build a chess engine to predict the next best move.

Path Finding Algorithm

My teammate and I built different agents using A* Algorithm that find path to the target in a given gridworld. Each agent is built for specific enviroments. Some of these environments are blindfolded agent, agent with moving target, gridworld with terrains where target is hidden. We also compared and analysed performance of these agents in terms of backtracking, pathlength and runtime.
